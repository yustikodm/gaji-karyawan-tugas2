# Tugas 2: Program Gaji Karyawan

Program Java untuk menghitung gaji karyawan dengan tunjangan dan bonus.

## Deskripsi
Program ini menghitung total gaji karyawan berdasarkan:
- Gaji pokok
- Tunjangan
- Bonus (jika ada)

## Cara Menjalankan
```bash
javac GajiKaryawan.java
java GajiKaryawan
```
